# Hangman

Welcome to Hangman! The computer selects a random word and the player has to guess the word letter by letter or all at once.

## Features

- Random word selection from a text file
- Can guess the word letter by letter or all at once
- Input validation
- Lives system with ASCII hangman drawing
- Play again feature

## How to Play

- The computer chooses a random word.
- Guess one letter at a time, or try to guess the entire word.
- Every incorrect guess removes one life and adds another part to the Hangman.
- Win by revealing the whole word before running out of lives.
